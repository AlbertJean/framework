Detailed Description

The public interface for OFCloudStorage allows you to upload and download data from the Open Feint server. Data is associated uniquely in 3 feilds, client application id, current user id, and a key passed in by the developer. This allows you to have static keys for your data in your application across all users (ex. static const NSString* saveGameDataKey = "SaveGameData")
